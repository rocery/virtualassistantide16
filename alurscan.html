Return

Return merupakan pengiriman paket yang tertunda, terhambat, berhenti, dan kembali dari pihak penerima, pihak ketiga maupun IDExpress

1

Kurir/Admin melakukan Scan Delivery AWB yang akan di Return

2

Scan Problem on Shipment (POS) dengan alasan yang sebenarnya

3

Hubungi Customer Exprience (CX) bahwa terdapat paket yang akan di Return

4

Creat Return Waybill di menu "Search Returned Packages (Center)" di OMS

5

Isi form dengan informasi AWB, Return Destination, Return Reason, Description, dan Bukti Foto jika ada, lalu Confirm

Alasan Paket Return:
  1. Penerima berada di luar kota dan tidak meminta penjadwalan ulang
  2. Data AWB rusak
  3. Penerima merasa tidak memesan paket atau menolak paket
  4. Penerima membatalkan pembelian
  5. Penerima tidak dapat dihubungi dan tidak terdapat penerima lain
  6. Kurir telah melakukan tiga kali percobaan pengiriman
  7. AWB ganda
  8. Penerima meminta Return
  9. AWB tertukar (Croslabel)
 
Confirm/Cancel Return:
  1. Konfirmasi ke seller apakah paket boleh di Return atau tidak
  2. Jika sudah konfirmasi seller, buka menu "Search Returned Packages (Branch)"
  3. Search AWB, klik AWB, Confirm/Cancel
 
Void Return

Void AWB Return dapat dilakukan apabila AWB sudah di confirm TH Origin dan hanya dapat dilakukan maksimal 24 jam setelah di confirm

Koordinator dapat melakukan void pada keadaan AWB sudah registrasi return tetapi ternyata customer datang ke TH untuk mengambil paket atau alamat customer ditemukan

  1. Login menggunakan akun Koordinator
  2. Buka menu "Search Returned Packages (Branch)"
  3. Search AWB, klik AWB, Void lalu Confirm
