Standar Operasional Prosedur (SOP)

Koordinator

Tugas
  • Monitoring paket yang tidak memenuhi standard packing
    • Fragile tanpa sticker/bubble wrap/packing kayu
    • Packing rusak/sobek/basah
    • Paket basah/bocor/pecah/patah
  • Daily report paket-paket bermasalah (dalam format excel)
  • Bertanggungjawab apabila ada paket yang rusak/bermasalah
  • Monitoring & kontrol status paket yang sudah dikirim ke MH apakah sesuai SLA
  • Monitoring paket missroute
  • Monitoring inventaris yang ada di TH
Tanggung Jawab
  • Briefing 15 menit sebelum kerja
    • Evaluasi dan motivasi kepada tim TH
    • Pengaturan dan pembagian tugas
    • Penyampaian informasi internal
    • Memastikan tim untuk selalu memakai atribut kerja sesuai peraturan
  • Monitoring scan arival dan scan sending
  • Monitoring seluruh pekerjaan apakah sesuai SOP dari awal sampai akhir
  • Memastikan dan mengontrol tim melakukan tugasnya sesuai dengan target yang ditentukan
  • Mengontrol dan monitor kebutuhan serta pemakaian perlengkapan operasional
  • Memastikan pemberangkatan seluruh paket sesuai dengan jadwal yang ditentukan
  • Wajib backup jika ada rekan koordinator lain berhalangan masuk kerja (sakit/izin/cuti

Admin

Tugas
  • Melayani Customer yang datang ke TH (Transit HUB)
  • Menerapkan SOP pengiriman paket dan repacking
  • Membuat laporan yang diminta oleh Finance, CX, HRD, dan Purchasing
  • Menugaskan kurir unutk melakukan Pickup paket
  • Handling Complaint ketika paket bermasalah
Penampilan Admin
  1. Menggunakan seragam sesuai penjanjian Kerja Bersama
  2. Menggunakan ID Card
  3. Rambut berwarna gelap/alami
  4. Tidak menindik hidung dan telinga bagi Pria
  5. Menggunakan sepatu
Repacking

Paket yang harus di repacking meliputi:

  1. Paket Frigile wajib menggunakan Bubble Wrap dan packing kayu jika cutomer menyetujui
  2. Paket dokumen harus menggunakan platik agar terhindar dari kerusakan (basah/sobek)
  3. Packing rusah (basah, sobek, berlubang) harus direpacking sebelum dikirimkan
Laporan
  1. Finance
    • Tagihan COD
    • Tagihan PAD
    • Tagihan Cash Pick Up
    • Data rekapan setoran (perminggu)
    • Data rekening koran (perminggu)
    • Reimburse, foto, scan, hardcopy per cut off
  2. Customer Expriece (CX)
    • Validate
    • Reason Late Pick Up
    • Raw Data Backlog
    • Data Return VIP
  3. HRD
    • Data Karyawan
    • Absesnsi
    • Data BPJS
    • Data BDO99
  4. Purchasing
    • Data POP TH

Processing

Tugas
  • Menjaga penampilan sesuai dengan SOP perusahaan
  • Monitoring alur paket mulai dari scan Pick Up sampai Sending ke MH
  • Monitoring alur Delivery dari MH sampai POD
  • Repacking paket jika terdapat kerusakan packing
  • Sorting paket sesuai dengan kategori (DG, Frozen Food, FragileP)
Dangerous Goods, Frozen Foods, Fragile
  1. Dangerous Goods (DG) adalah bahan atau zat yang memiliki potensi untuk membahayakan keselamatan, kesehatan, maupun harta
    • Class 1 : Mudah meledak (peluru, kembang api)
    • Class 2 : Gas (korek api, gas kompor, deodorant)
    • Class 3 : Gas mudah terbakar (cat, miras, tiner, parfum)
    • Class 4 : Benda mudah terbakar (posfor, kalsium, karbit)
    • Class 5 : Zat oksidator (pupuk amonium nitrat
  2. Frozen Foods adalah paket yang berisi makan beku yang perlu segera dikirimkan supaya tidak membusuk, rusah, dll
  3. Fragile merupakan paket yang mudah pecah (keramik, kaca, peralatan elektronik)
Lain-lain
  1. Monitoring data Pick Up dan Delivery bekerjasama dengan Koordinator dan Admin
  2. Menjaga kebersihan TH
  3. Tidak merokok di TH

Kurir

Tugas
  • Menjaga penampilan sesuai dengan SOP perusahaan
  • Penyusunan paket pada karung/tas ID Express menyesuaikan jarak, berat paket, dan jenis paket
  • Mengerti alur scan Pick Up sampai sending ke MH
  • Mengerti alur scan Delivery sampai POD, apabila gagal terkirim segera POS
  • Repacking paket jika terdapat kerusakan paket
  • Mematuhi SOP dan arahan Koordinator
Penampilan Kurir
  • Menggunakan seragam ID Express (Jaket, Kaus Polo, Helm, ID Card)
  • Menggunakan celana panjang berwarna gelap
  • Menggunakan sepatu
  • Rambut rapi dan tidak diwarnai (warna alami)
  • Tidak menggunakan tindik
  • Jika memiliki tato wajib ditutupi
Penyusunan Paket
  1. Bagian atas karung diisi paket dengan alamat paling dekat dari TH atau diisi paket Fragile/Dokumen
  2. Bagian tengah karung diisi paket dengan alamat yang dinilai cukup jauh dari TH
  3. Bagian bawah karung diisi paket dengan alamat paling jauh dari TH atau diisi paket berat/besar
Lain-Lain
  • Kurir diharuskan menjaga kebersihan TH
  • Dilarang merokok di TH
  • Menggunakan bahasa sopan dan santun ketika berkomunikasi dengan Customer
  • Bila mengalami permasalah dilapangan segera lapor Koordinator/Admin
